## 5.1 Agent系统开发

### 🎯 Agent核心概念

```
┌─────────────────────────────────────────────────────────┐
│         Agent：自主决策与行动的智能系统                   │
└─────────────────────────────────────────────────────────┘

核心定义：
┌─────────────────────────────────┐
│  Agent = LLM + 规划 + 工具 + 记忆│
│                                 │
│  特点：                         │
│  - 自主性：能够自主决策          │
│  - 反应性：能够感知环境          │
│  - 主动性：能够主动行动          │
│  - 社交性：能够与其他Agent交互   │
│                                 │
│  与传统LLM的区别：              │
│  - LLM：被动回答问题            │
│  - Agent：主动规划、执行、反思  │
│                                 │
│  例如：                         │
│  用户："帮我写一篇关于AI的文章" │
│                                 │
│  LLM：直接生成文章              │
│                                 │
│  Agent：                        │
│  1. 规划：分解任务              │
│     - 搜索相关资料              │
│     - 分析关键观点              │
│     - 构建文章结构              │
│     - 撰写文章内容              │
│  2. 执行：调用工具              │
│     - 使用搜索引擎              │
│     - 使用文本分析工具          │
│     - 使用写作工具              │
│  3. 反思：检查结果              │
│     - 检查内容质量              │
│     - 检查逻辑连贯              │
│     - 修改完善                  │
└─────────────────────────────────┘

Agent架构：
┌─────────────────────────────────┐
│  ┌───────────────────────┐     │
│  │  用户任务              │     │
│  └───────────────────────┘     │
│           ↓                     │
│  ┌───────────────────────┐     │
│  │  规划模块              │     │
│  │  - 任务分解            │     │
│  │  - 生成计划            │     │
│  │  - 选择策略            │     │
│  └───────────────────────┘     │
│           ↓                     │
│  ┌───────────────────────┐     │
│  │  执行模块              │     │
│  │  - 工具选择            │     │
│  │  - 工具调用            │     │
│  │  - 结果处理            │     │
│  └───────────────────────┘     │
│           ↓                     │
│  ┌───────────────────────┐     │
│  │  反思模块              │     │
│  │  - 结果评估            │     │
│  │  - 错误纠正            │     │
│  │  - 计划调整            │     │
│  └───────────────────────┘     │
│           ↓                     │
│  ┌───────────────────────┐     │
│  │  记忆模块              │     │
│  │  - 短期记忆            │     │
│  │  - 长期记忆            │     │
│  │  - 知识积累            │     │
│  └───────────────────────┘     │
│           ↓                     │
│  ┌───────────────────────┐     │
│  │  输出结果              │     │
│  └───────────────────────┘     │
└─────────────────────────────────┘
```

### 📊 Agent设计模式

```
┌─────────────────────────────────────────────────────────┐
│         四种主流Agent设计模式                             │
└─────────────────────────────────────────────────────────┘

1. ReAct模式（Reasoning + Acting）
┌─────────────────────────────────┐
│  原理：交替推理和行动           │
│                                 │
│  流程：                         │
│  Thought：我需要搜索相关信息     │
│  Action：调用搜索工具            │
│  Observation：搜索结果           │
│  Thought：我需要分析这些信息     │
│  Action：调用分析工具            │
│  Observation：分析结果           │
│  ...                            │
│  最终答案：综合所有信息          │
│                                 │
│  示例：                         │
│  任务："找出2024年AI领域的重大突破"│
│                                 │
│  Thought 1：我需要搜索2024年AI新闻│
│  Action 1：Search["2024 AI breakthrough"]│
│  Observation 1：找到10篇相关文章 │
│                                 │
│  Thought 2：我需要分析这些文章   │
│  Action 2：Analyze[articles]    │
│  Observation 2：发现3个主要突破  │
│                                 │
│  Thought 3：我需要整理这些突破   │
│  Action 3：Summarize[breakthroughs]│
│  Observation 3：整理完成         │
│                                 │
│  Answer：列出3个重大突破         │
│                                 │
│  优势：                         │
│  - 明确的推理过程                │
│  - 可解释性强                    │
│  - 易于调试                      │
└─────────────────────────────────┘

2. Plan-and-Execute模式
┌─────────────────────────────────┐
│  原理：先规划，后执行            │
│                                 │
│  流程：                         │
│  步骤1：规划                     │
│  - 分析任务                      │
│  - 制定完整计划                  │
│  - 生成步骤列表                  │
│                                 │
│  步骤2：执行                     │
│  - 按计划逐步执行                │
│  - 调用相应工具                  │
│  - 处理执行结果                  │
│                                 │
│  步骤3：验证                     │
│  - 检查执行结果                  │
│  - 验证目标达成                  │
│  - 修改完善                      │
│                                 │
│  示例：                         │
│  任务："写一份市场调研报告"      │
│                                 │
│  规划：                         │
│  1. 确定调研主题和范围           │
│  2. 搜索市场数据和趋势           │
│  3. 分析竞争对手                 │
│  4. 识别关键机会和风险           │
│  5. 撰写报告                     │
│                                 │
│  执行：                         │
│  步骤1：确定主题为"AI市场"       │
│  步骤2：搜索AI市场数据           │
│  步骤3：分析主要AI公司           │
│  步骤4：总结机会和风险           │
│  步骤5：撰写完整报告             │
│                                 │
│  优势：                         │
│  - 结构化执行                    │
│  - 易于并行                      │
│  - 目标明确                      │
└─────────────────────────────────┘

3. Multi-Agent模式
┌─────────────────────────────────┐
│  原理：多个Agent协作             │
│                                 │
│  角色：                         │
│  - 研究Agent：负责信息收集       │
│  - 分析Agent：负责数据分析       │
│  - 写作Agent：负责内容撰写       │

│  研究Agent → 收集信息           │

│  分析Agent → 分析数据           │
│      ↓                          │

4. Self-Reflection模式
```

### 💻 Agent系统实现

```python
from typing import Dict, List, Any
import json
from abc import ABC, abstractmethod

class Tool(ABC):
    @abstractmethod
    def name(self) -> str:
        pass
    
    @abstractmethod
    def description(self) -> str:
        pass
    
    @abstractmethod
    def run(self, input: Any) -> Any:
        pass

class SearchTool(Tool):
    def name(self) -> str:
        return "search"
    
    def description(self) -> str:
        return "搜索网络信息，输入查询关键词，返回搜索结果"
    
    def run(self, query: str) -> List[Dict]:
        return [
            {'title': f'搜索结果1: {query}', 'content': '相关内容1'},
            {'title': f'搜索结果2: {query}', 'content': '相关内容2'}
        ]

class AnalyzeTool(Tool):
    def name(self) -> str:
        return "analyze"
    
    def description(self) -> str:
        return "分析文本内容，输入文本，返回分析结果"
    
    def run(self, text: str) -> Dict:
        return {
            'summary': f'分析摘要',
            'key_points': ['要点1', '要点2'],
            'sentiment': 'positive'
        }

class WriteTool(Tool):
    def name(self) -> str:
        return "write"
    
    def description(self) -> str:
        return "撰写文本内容，输入主题和要求，返回撰写的文本"
    
    def run(self, topic: str, requirements: List[str]) -> str:
        return f"关于{topic}的文章内容"

class AgentMemory:
    def __init__(self):
        self.short_term_memory = []
        self.long_term_memory = {}
    
    def add_short_term(self, thought: str, action: str, observation: str):
        self.short_term_memory.append({
            'thought': thought,
            'action': action,
            'observation': observation
        })
    
    def add_long_term(self, key: str, value: Any):
        self.long_term_memory[key] = value
    
    def get_short_term(self) -> List[Dict]:
        return self.short_term_memory
    
    def get_long_term(self, key: str) -> Any:
        return self.long_term_memory.get(key)
    
    def clear_short_term(self):
        self.short_term_memory = []

class ReActAgent:
    def __init__(self, llm_client, tools: List[Tool]):
        self.llm_client = llm_client
        self.tools = {tool.name(): tool for tool in tools}
        self.memory = AgentMemory()
    
    def plan(self, task: str) -> str:
        prompt = f"""分析以下任务，制定执行计划：

任务：{task}

可用工具：
{chr(10).join(f'- {name}: {tool.description()}' for name, tool in self.tools.items())}

请制定详细的执行计划，包括：
1. 任务分解
2. 工具选择
3. 执行顺序

计划："""
        
        response = self.llm_client.chat.completions.create(
            model="gpt-4",
            messages=[{"role": "user", "content": prompt}],
            temperature=0.5
        )
        
        return response.choices[0].message.content
    
    def think(self, task: str, history: List[Dict]) -> str:
        history_text = "\n".join([
            f"Thought: {step['thought']}\nAction: {step['action']}\nObservation: {step['observation']}"
            for step in history
        ])
        
        prompt = f"""任务：{task}

历史记录：
{history_text if history_text else "无"}

可用工具：
{chr(10).join(f'- {name}: {tool.description()}' for name, tool in self.tools.items())}

下一步思考：
Thought："""
        
        response = self.llm_client.chat.completions.create(
            model="gpt-4",
            messages=[{"role": "user", "content": prompt}],
            temperature=0.5
        )
        
        return response.choices[0].message.content
    
    def act(self, thought: str) -> Dict:
        prompt = f"""根据以下思考，选择要执行的行动：

思考：{thought}

可用工具：
{chr(10).join(f'- {name}: {tool.description()}' for name, tool in self.tools.items())}

请以JSON格式输出：
{
    "tool": "工具名称",
    "input": "工具输入"
}

Action："""
        
        response = self.llm_client.chat.completions.create(
            model="gpt-4",
            messages=[{"role": "user", "content": prompt}],
            temperature=0.3
        )
        
        try:
            action = json.loads(response.choices[0].message.content)
        except:
            action = {'tool': 'none', 'input': ''}
        
        return action
    
    def observe(self, action: Dict) -> str:
        tool_name = action.get('tool')
        tool_input = action.get('input')
        
        if tool_name in self.tools:
            result = self.tools[tool_name].run(tool_input)
            return json.dumps(result, ensure_ascii=False)
        
        return "无工具调用"
    
    def reflect(self, task: str, history: List[Dict]) -> str:
        history_text = "\n".join([
            f"Thought: {step['thought']}\nAction: {step['action']}\nObservation: {step['observation']}"
            for step in history
        ])
        
        prompt = f"""任务：{task}

执行历史：
{history_text}

请评估执行结果：
1. 任务是否完成？
2. 结果质量如何？
3. 是否需要改进？

评估："""
        
        response = self.llm_client.chat.completions.create(
            model="gpt-4",
            messages=[{"role": "user", "content": prompt}],
            temperature=0.5
        )
        
        return response.choices[0].message.content
    
    def run(self, task: str, max_iterations: int = 10) -> Dict:
        plan = self.plan(task)
        
        self.memory.clear_short_term()
        
        iteration = 0
        while iteration < max_iterations:
            thought = self.think(task, self.memory.get_short_term())
            
            if "任务完成" in thought or "Final Answer:" in thought:
                break
            
            action = self.act(thought)
            
            observation = self.observe(action)
            
            self.memory.add_short_term(thought, json.dumps(action), observation)
            
            iteration += 1
        
        reflection = self.reflect(task, self.memory.get_short_term())
        
        final_answer = self.extract_final_answer(task)
        
        return {
            'task': task,
            'plan': plan,
            'history': self.memory.get_short_term(),
            'reflection': reflection,
            'answer': final_answer
        }
    
    def extract_final_answer(self, task: str) -> str:
        history = self.memory.get_short_term()
        
        history_text = "\n".join([
            f"Thought: {step['thought']}\nAction: {step['action']}\nObservation: {step['observation']}"
            for step in history
        ])
        
        prompt = f"""任务：{task}

执行历史：
{history_text}

请给出最终答案："""
        
        response = self.llm_client.chat.completions.create(
            model="gpt-4",
            messages=[{"role": "user", "content": prompt}],
            temperature=0.5
        )
        
        return response.choices[0].message.content

class MultiAgentSystem:
    def __init__(self, llm_client):
        self.llm_client = llm_client
        self.agents = {}
    
    def create_agent(self, name: str, role: str, tools: List[Tool]):
        agent = ReActAgent(self.llm_client, tools)
        agent.role = role
        self.agents[name] = agent
    
    def coordinate(self, task: str) -> Dict:
        prompt = f"""分析任务，分配给合适的Agent：

任务：{task}

可用Agent：
{chr(10).join(f'- {name}: {agent.role}' for name, agent in self.agents.items())}

请以JSON格式输出分配方案：
{
    "workflow": [
        {
            "agent": "Agent名称",
            "task": "分配的任务",
            "dependencies": ["依赖的前置任务"]
        }
    ]
}

分配方案："""
        
        response = self.llm_client.chat.completions.create(
            model="gpt-4",
            messages=[{"role": "user", "content": prompt}],
            temperature=0.5
        )
        
        try:
            workflow = json.loads(response.choices[0].message.content)
        except:
            workflow = {'workflow': []}
        
        return workflow
    
    def execute_workflow(self, task: str) -> Dict:
        workflow = self.coordinate(task)
        
        results = {}
        
        for step in workflow['workflow']:
            agent_name = step['agent']
            sub_task = step['task']
            dependencies = step.get('dependencies', [])
            
            if dependencies:
                dependency_results = {
                    dep: results.get(dep) for dep in dependencies
                }
                sub_task = f"{sub_task}\n前置任务结果：{json.dumps(dependency_results, ensure_ascii=False)}"
            
            if agent_name in self.agents:
                result = self.agents[agent_name].run(sub_task)
                results[agent_name] = result
        
        final_result = self.synthesize_results(task, results)
        
        return {
            'task': task,
            'workflow': workflow,
            'agent_results': results,
            'final_result': final_result
        }
    
    def synthesize_results(self, task: str, results: Dict) -> str:
        results_text = "\n".join([
            f"{name}: {result['answer']}"
            for name, result in results.items()
        ])
        
        prompt = f"""综合各Agent的结果，给出最终答案：

任务：{task}

各Agent结果：
{results_text}

最终答案："""
        
        response = self.llm_client.chat.completions.create(
            model="gpt-4",
            messages=[{"role": "user", "content": prompt}],
            temperature=0.5
        )
        
        return response.choices[0].message.content

def test_agent():
    from openai import OpenAI
    
    client = OpenAI(api_key="your-api-key")
    
    tools = [
        SearchTool(),
        AnalyzeTool(),
        WriteTool()
    ]
    
    agent = ReActAgent(client, tools)
    
    task = "研究2024年AI领域的重大突破，并写一份简要报告"
    
    result = agent.run(task)
    
    print(f"任务：{result['task']}")
    print(f"计划：{result['plan']}")
    print(f"\n执行历史：")
    for i, step in enumerate(result['history']):
        print(f"步骤{i+1}：")
        print(f"  Thought: {step['thought']}")
        print(f"  Action: {step['action']}")
        print(f"  Observation: {step['observation']}")
    print(f"\n反思：{result['reflection']}")
    print(f"\n最终答案：{result['answer']}")

if __name__ == "__main__":
    test_agent()
```

```python
from typing import Dict, List, Tuple
import json

class KnowledgeGraph:
    def __init__(self):
        self.entities = {}
        self.relations = {}
        self.triples = []
    
    def add_entity(self, entity_id: str, entity_type: str, attributes: Dict):
        self.entities[entity_id] = {
            'type': entity_type,
            'attributes': attributes
        }
    
    def add_relation(self, relation_name: str, relation_type: str):
        self.relations[relation_name] = {
            'type': relation_type
        }
    
    def add_triple(self, subject: str, relation: str, object: str):
        self.triples.append({
            'subject': subject,
            'relation': relation,
            'object': object
        })
    
    def query_entity(self, entity_id: str) -> Dict:
        entity_info = self.entities.get(entity_id, {})
        
        related_triples = [
            t for t in self.triples
            if t['subject'] == entity_id or t['object'] == entity_id
        ]
        
        return {
            'entity': entity_info,
            'relations': related_triples
        }
    
    def query_relation(self, subject: str, relation: str) -> List[str]:
        results = [
            t['object'] for t in self.triples
            if t['subject'] == subject and t['relation'] == relation
        ]
        
        return results
    
    def query_path(self, start_entity: str, end_entity: str, max_depth: int = 3) -> List[List[str]]:
        paths = []
        
        def dfs(current: str, target: str, path: List[str], depth: int):
            if depth > max_depth:
                return
            
            if current == target:
                paths.append(path.copy())
                return
            
            related = [
                t for t in self.triples
                if t['subject'] == current
            ]
            
            for triple in related:
                next_entity = triple['object']
                if next_entity not in path:
                    path.append(f"{triple['relation']} → {next_entity}")
                    dfs(next_entity, target, path, depth + 1)
                    path.pop()
        
        dfs(start_entity, end_entity, [start_entity], 0)
        
        return paths
    
    def infer_relation(self, subject: str, relation: str) -> List[str]:
        direct_results = self.query_relation(subject, relation)
        
        if direct_results:
            return direct_results
        
        inferred_results = []
        
        for triple in self.triples:
            if triple['subject'] == subject:
                intermediate = triple['object']
                inferred = self.query_relation(intermediate, relation)
                inferred_results.extend(inferred)
        
        return inferred_results
```

```python
from typing import Dict, List, Any, Callable
import json

class ToolRegistry:
    def __init__(self):
        self.tools = {}
    
    def register_tool(self, name: str, function: Callable, description: str, parameters: Dict):
        self.tools[name] = {
            'function': function,
            'description': description,
            'parameters': parameters
        }
    
    def get_tool(self, name: str) -> Dict:
        return self.tools.get(name)
    
    def list_tools(self) -> List[Dict]:
        return [
            {
                'name': name,
                'description': tool['description'],
                'parameters': tool['parameters']
            }
            for name, tool in self.tools.items()
        ]

class ToolCallingAgent:
    def __init__(self, llm_client, tool_registry: ToolRegistry):
        self.llm_client = llm_client
        self.tool_registry = tool_registry
    
    def analyze_request(self, user_request: str) -> Dict:
        tools_info = self.tool_registry.list_tools()
        
        tools_text = json.dumps(tools_info, ensure_ascii=False, indent=2)
        
        prompt = f"""分析用户请求，决定是否需要调用工具：

用户请求：{user_request}

可用工具：
{tools_text}

请以JSON格式输出：
{
    "need_tools": true/false,
    "tools_to_call": [
        {
            "name": "工具名称",
            "parameters": {
                "参数名": "参数值"
            }
        }
    ],
    "reason": "调用原因"
}

分析："""
        
        response = self.llm_client.chat.completions.create(
            model="gpt-4",
            messages=[{"role": "user", "content": prompt}],
            temperature=0.3
        )
        
        try:
            analysis = json.loads(response.choices[0].message.content)
        except:
            analysis = {'need_tools': False, 'tools_to_call': [], 'reason': ''}
        
        return analysis
    
    def call_tool(self, tool_name: str, parameters: Dict) -> Any:
        tool = self.tool_registry.get_tool(tool_name)
        
        if tool:
            return tool['function'](**parameters)
        
        return None
    
    def process_request(self, user_request: str) -> Dict:
        analysis = self.analyze_request(user_request)
        
        tool_results = []
        
        if analysis['need_tools']:
            for tool_call in analysis['tools_to_call']:
                tool_name = tool_call['name']
                parameters = tool_call['parameters']
                
                result = self.call_tool(tool_name, parameters)
                
                tool_results.append({
                    'tool': tool_name,
                    'parameters': parameters,
                    'result': result
                })
        
        final_response = self.generate_response(user_request, tool_results)
        
        return {
            'user_request': user_request,
            'analysis': analysis,
            'tool_results': tool_results,
            'response': final_response
        }
    
    def generate_response(self, user_request: str, tool_results: List[Dict]) -> str:
        if not tool_results:
            prompt = f"""回答用户请求：

用户请求：{user_request}

回答："""
        else:
            results_text = json.dumps(tool_results, ensure_ascii=False, indent=2)

用户请求：{user_request}

工具调用结果：
{results_text}

回答："""
        
        response = self.llm_client.chat.completions.create(
            model="gpt-4",

            temperature=0.7
        )
```

```python
            perception = {

                'opportunities': [],
                'threats': [],
                'current_state': ''
            }
        
        self.current_state = perception

感知结果：

历史决策：
{json.dumps(self.decision_history[-5:], ensure_ascii=False, indent=2)}

请以JSON格式输出决策：

决策："""

决策：

请生成具体的执行计划，以JSON格式输出：
{
    "actions": [

            "action": "具体行动",
            "parameters": {}
        }
    ],
    "timeline": "执行时间",
    "resources": ["所需资源"]
}

执行计划："""

            model="gpt-4",
            messages=[{"role": "user", "content": prompt}],
            temperature=0.5
        )
        
        try:
            execution_plan = json.loads(response.choices[0].message.content)
        except:
            execution_plan = {'actions': [], 'timeline': '', 'resources': []}
        
        return execution_plan
    
    def evaluate(self, decision: Dict, outcome: Dict) -> Dict:
        prompt = f"""评估决策结果：

决策：
{json.dumps(decision, ensure_ascii=False, indent=2)}

实际结果：
{json.dumps(outcome, ensure_ascii=False, indent=2)}

请评估决策效果，以JSON格式输出：
{
    "success": true/false,
    "achievement": "达成程度",
    "lessons": ["经验教训"],
    "improvements": ["改进建议"]
}

评估结果："""
        
        response = self.llm_client.chat.completions.create(
            model="gpt-4",
            messages=[{"role": "user", "content": prompt}],
            temperature=0.5
        )
        
        try:
            evaluation = json.loads(response.choices[0].message.content)
        except:
            evaluation = {
                'success': False,
                'achievement': '',
                'lessons': [],
                'improvements': []
            }
        
        return evaluation

        prompt = f"""从评估结果中学习：

评估结果：
{json.dumps(evaluation, ensure_ascii=False, indent=2)}

请总结学习内容，以JSON格式输出：
{
    "knowledge_gained": ["获得的知识"],
    "strategy_updates": ["策略更新"],
    "future_recommendations": ["未来建议"]

        response = self.llm_client.chat.completions.create(
            model="gpt-4",
            messages=[{"role": "user", "content": prompt}],
            temperature=0.5
        )

            learning = json.loads(response.choices[0].message.content)

        perception = self.perceive(environment)

        simulated_outcome = self.simulate_execution(execution_plan)

        learning = self.learn(evaluation)
        
        return {
            'perception': perception,
            'decision': decision,
            'execution_plan': execution_plan,
            'outcome': simulated_outcome,
            'evaluation': evaluation,
            'learning': learning
        }
    
    def simulate_execution(self, execution_plan: Dict) -> Dict:
        return {
            'status': 'completed',
            'results': ['执行结果1', '执行结果2'],
            'metrics': {'efficiency': 0.9, 'quality': 0.85}
        }
```

```python
        self.coordinator = CoordinatorAgent(llm_client)
        self.searcher = SearchAgent(llm_client)
        self.analyzer = AnalysisAgent(llm_client)
        self.writer = WritingAgent(llm_client)
        self.reviewer = ReviewAgent(llm_client)
    
    def conduct_research(self, research_topic: str) -> Dict:
        task_breakdown = self.coordinator.breakdown_task(research_topic)
        
        search_results = {}
        for subtask in task_breakdown['search_tasks']:
            search_results[subtask['id']] = self.searcher.search(subtask['query'])
        
        analysis_results = {}
        for subtask in task_breakdown['analysis_tasks']:
            data = search_results.get(subtask['data_source'], [])
            analysis_results[subtask['id']] = self.analyzer.analyze(data, subtask['analysis_type'])
        
        writing_results = {}
        for subtask in task_breakdown['writing_tasks']:
            content_data = {
                'search_results': search_results,
                'analysis_results': analysis_results
            }
            writing_results[subtask['id']] = self.writer.write(subtask['section'], content_data)
        
        report = self.writer.compile_report(writing_results)
        
        review_result = self.reviewer.review(report)
        
        if review_result['needs_revision']:
            report = self.writer.revise(report, review_result['suggestions'])
        
        return {
            'research_topic': research_topic,
            'task_breakdown': task_breakdown,
            'search_results': search_results,
            'analysis_results': analysis_results,
            'writing_results': writing_results,
            'report': report,
            'review': review_result
```
