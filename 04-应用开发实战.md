## 4.1 Prompt工程高级技巧

### 🎯 Prompt设计原则

```
┌─────────────────────────────────────────────────────────┐
│         Prompt工程：与LLM有效沟通的艺术                   │
└─────────────────────────────────────────────────────────┘

核心原则：
┌─────────────────────────────────┐
│  1. 清晰明确                    │
│  - 明确任务目标                 │
│  - 具体的输出要求               │
│  - 避免模糊表述                 │
│                                 │
│  2. 提供上下文                  │
│  - 相关背景信息                 │
│  - 任务约束条件                 │
│  - 目标受众                     │
│                                 │
│  3. 给出示例                    │
│  - 输入输出示例                 │
│  - 格式参考                     │
│  - 少样本学习                   │
│                                 │
│  4. 分步引导                    │
│  - 复杂任务分解                 │
│  - 步骤化指导                   │
│  - 思维链                       │
│                                 │
│  5. 指定格式                    │
│  - 输出格式要求                 │
│  - 结构化输出                   │
│  - JSON、表格等                 │
└─────────────────────────────────┘

对比示例：
┌─────────────────────────────────┐
│  ❌ 不好的Prompt：              │
│  "写一篇文章"                   │
│                                 │
│  问题：                         │
│  - 没有主题                     │
│  - 没有长度要求                 │
│  - 没有风格要求                 │
│  - 模型输出不可控               │
│                                 │
│  ✅ 好的Prompt：                │
│  "写一篇500字的文章，主题是人工 │
│  智能对未来工作的影响。要求：   │
│  1. 目标读者是大学生            │
│  2. 语言通俗易懂                │
│  3. 包含3个具体案例             │
│  4. 最后给出建议                │
│  5. 以小标题分段"               │
│                                 │
│  优势：                         │
│  - 明确主题和长度               │
│  - 指定读者和风格               │
│  - 要求具体内容                 │
│  - 模型输出可控                 │
└─────────────────────────────────┘
```

### 📊 高级Prompt技巧

```
┌─────────────────────────────────────────────────────────┐
│         五种高级Prompt技巧                               │
└─────────────────────────────────────────────────────────┘

1. 角色扮演（Role Prompting）
┌─────────────────────────────────┐
│  原理：赋予模型特定角色         │
│                                 │
│  示例：                         │
│  "你是一位资深的Python开发工程师，│
│   有10年的开发经验。你擅长：    │
│   - 编写高质量、可维护的代码    │
│   - 性能优化和调试              │
│   - 设计模式和最佳实践          │
│                                 │
│   请以这个身份回答用户的问题，确保：│
│   1. 代码符合PEP 8规范          │
│   2. 考虑边界情况和错误处理     │
│   3. 提供清晰的注释             │
│   4. 给出性能优化建议"          │
│                                 │
│  效果：                         │
│  - 模型输出更专业               │
│  - 符合角色特点                 │
│  - 提高输出质量                 │
└─────────────────────────────────┘

2. 思维链（Chain of Thought）
┌─────────────────────────────────┐
│  原理：引导模型逐步推理         │
│                                 │
│  示例：                         │
│  "问题：小明有5个苹果，给了小红2个，│
│   又买了3个，现在有几个？       │
│                                 │
│   请一步步思考：                │
│   1. 小明最初有几个苹果？答：5个│
│   2. 给了小红几个？答：2个      │
│   3. 这时还剩几个？答：5-2=3个  │
│   4. 又买了几个？答：3个        │
│   5. 现在有几个？答：3+3=6个    │
│                                 │
│   最终答案：6个苹果"            │
│                                 │
│  效果：                         │
│  - 提高复杂问题准确率           │
│  - 减少推理错误                 │
│  - 输出更可解释                 │
└─────────────────────────────────┘

3. 少样本学习（Few-shot Learning）
┌─────────────────────────────────┐
│  原理：提供示例引导模型         │
│                                 │
│  示例：                         │
│  "任务：判断情感倾向            │
│                                 │
│  示例1：                        │
│  文本：这部电影太精彩了，我看了三遍！│
│  情感：正面                     │
│                                 │
│  示例2：                        │
│  文本：服务态度很差，再也不来了。│
│  情感：负面                     │
│                                 │
│  示例3：                        │
│  文本：今天天气一般，不好也不坏。│
│  情感：中性                     │
│                                 │
│  现在请判断：                    │
│  文本：这个产品性价比很高，推荐购买。│
│  情感："                        │
│                                 │
│  效果：                         │
│  - 模型快速学习任务             │
│  - 输出格式一致                 │
│  - 提高准确率                   │
└─────────────────────────────────┘

4. 自我反思（Self-Reflection）
┌─────────────────────────────────┐
│  原理：让模型检查和改进自己     │
│                                 │
│  示例：                         │
│  "请完成以下任务：              │
│  1. 首先给出你的答案            │
│  2. 然后检查你的答案是否正确    │
│  3. 如果发现错误，请修正        │
│  4. 最后给出最终答案            │
│                                 │
│  问题：[你的问题]"              │
│                                 │
│  流程：                         │
│  步骤1：模型给出初步答案        │
│  步骤2：模型自我检查            │
│  步骤3：模型发现错误并修正      │
│  步骤4：模型给出最终答案        │
│                                 │
│  效果：                         │
│  - 提高答案质量                 │
│  - 减少错误                     │
│  - 增强可靠性                   │
└─────────────────────────────────┘

5. 结构化输出（Structured Output）
┌─────────────────────────────────┐
│  原理：要求模型输出特定格式     │
│                                 │
│  示例：                         │
│  "请以JSON格式输出分析结果，包含以下字段：│
│  {                              │
│    'summary': '问题总结',       │
│    'key_points': ['要点1', '要点2', '要点3'],│
│    'recommendations': [         │
│      {                          │
│        'action': '建议行动',    │
│        'priority': '高/中/低',  │
│        'rationale': '理由'      │
│      }                          │
│    ],                           │
│    'risks': ['风险1', '风险2']  │
│  }"                             │
│                                 │
│  效果：                         │
│  - 输出格式统一                 │
│  - 易于程序处理                 │
│  - 提高可靠性                   │
└─────────────────────────────────┘
```

```python
class PromptTemplates:
    @staticmethod
    def role_prompt(role, skills, requirements):
        template = f"""你是一位{role}。

你的技能包括：
{chr(10).join(f'- {skill}' for skill in skills)}

请以这个身份回答用户的问题，确保：
{chr(10).join(f'{i+1}. {req}' for i, req in enumerate(requirements))}

用户问题："""
        return template
    
    @staticmethod
    def chain_of_thought_prompt(problem, steps=None):
        if steps:
            steps_text = chr(10).join(f'{i+1}. {step}' for i, step in enumerate(steps))
        else:
            steps_text = "请一步步思考"
        
        template = f"""问题：{problem}

{steps_text}

最终答案："""
        return template
    
    @staticmethod
    def few_shot_prompt(task_description, examples, new_input):
        examples_text = ""
        for i, example in enumerate(examples):
            examples_text += f"""示例{i+1}：
输入：{example['input']}
输出：{example['output']}
{chr(10)}"""
        
        template = f"""任务：{task_description}

{examples_text}
现在请处理：
输入：{new_input}
输出："""
        return template
    
    @staticmethod
    def structured_output_prompt(task, schema):
        schema_text = json.dumps(schema, indent=2, ensure_ascii=False)
        
        template = f"""任务：{task}

请以以下JSON格式输出：
{schema_text}

确保：
1. 输出是有效的JSON
2. 所有字段都填写
3. 内容准确完整

输入："""
        return template
    
    @staticmethod
    def self_reflection_prompt(task, problem):
        template = f"""任务：{task}

请按以下步骤完成：
1. 首先给出你的答案
2. 然后检查你的答案是否正确
3. 如果发现错误，请修正
4. 最后给出最终答案

问题：{problem}

步骤1：初步答案
"""
        return template

def test_prompt_templates():

    role_prompt = templates.role_prompt(
        role="资深Python开发工程师",
        skills=["编写高质量代码", "性能优化", "调试"],
        requirements=["代码符合PEP 8", "考虑边界情况", "提供注释"]
    )
    print("角色扮演Prompt：")
    print(role_prompt)
    print("\n" + "="*50 + "\n")
    
    cot_prompt = templates.chain_of_thought_prompt(
        problem="小明有5个苹果，给了小红2个，又买了3个，现在有几个？",
        steps=["小明最初有几个苹果？", "给了小红几个？", "还剩几个？", "又买了几个？", "现在有几个？"]
    )
    print("思维链Prompt：")
    print(cot_prompt)
```

## 4.2 RAG系统开发

### 🎯 RAG核心原理

```
┌─────────────────────────────────────────────────────────┐
│         RAG：检索增强生成                                │
└─────────────────────────────────────────────────────────┘

核心思想：
┌─────────────────────────────────┐
│  问题：LLM的局限性              │
│  - 知识截止日期                 │
│  - 无法访问私有数据             │
│  - 可能产生幻觉                 │
│                                 │
│  解决：RAG                      │
│  - 检索：从知识库获取相关信息   │
│  - 增强：将信息作为上下文       │
│  - 生成：基于上下文生成回答     │
│                                 │
│  流程：                         │
│  用户问题 → 向量化 → 检索 →    │
│  获取文档 → 构建Prompt →       │
│  LLM生成 → 回答                 │
└─────────────────────────────────┘

RAG架构：
┌─────────────────────────────────┐
│  ┌───────────────────────┐     │
│  │  用户问题              │     │
│  └───────────────────────┘     │
│           ↓                     │
│  ┌───────────────────────┐     │
│  │  向量化                │     │
│  │  问题 → 向量           │     │
│  └───────────────────────┘     │
│           ↓                     │
│  ┌───────────────────────┐     │
│  │  向量检索              │     │
│  │  在向量数据库中搜索     │     │
│  │  找到最相似的文档       │     │
│  └───────────────────────┘     │
│           ↓                     │
│  ┌───────────────────────┐     │
│  │  文档获取              │     │
│  │  获取相关文档内容       │     │
│  │  可能重排序             │     │
│  └───────────────────────┘     │
│           ↓                     │
│  ┌───────────────────────┐     │
│  │  Prompt构建            │     │
│  │  问题 + 文档作为上下文 │     │
│  └───────────────────────┘     │
│           ↓                     │
│  ┌───────────────────────┐     │
│  │  LLM生成               │     │
│  │  基于上下文生成回答     │     │
│  └───────────────────────┘     │
│           ↓                     │
│  ┌───────────────────────┐     │
│  │  输出回答              │     │
│  └───────────────────────┘     │
└─────────────────────────────────┘
```

### 💻 完整RAG实现

```python
import torch
from sentence_transformers import SentenceTransformer
import faiss
import numpy as np
from typing import List, Dict
import json

class VectorStore:
    def __init__(self, embedding_model="all-MiniLM-L6-v2"):
        self.encoder = SentenceTransformer(embedding_model)
        self.documents = []
        self.metadata = []
        self.index = None
    
    def add_documents(self, documents: List[Dict]):
        texts = [doc['content'] for doc in documents]
        
        embeddings = self.encoder.encode(texts)
        
        self.documents.extend(texts)
        self.metadata.extend([doc.get('metadata', {}) for doc in documents])
        
        dimension = embeddings.shape[1]
        if self.index is None:
            self.index = faiss.IndexFlatL2(dimension)
        
        self.index.add(embeddings.astype('float32'))
    
    def search(self, query: str, k: int = 5) -> List[Dict]:
        query_embedding = self.encoder.encode([query])
        
        distances, indices = self.index.search(query_embedding.astype('float32'), k)
        
        results = []
        for i, idx in enumerate(indices[0]):
            if idx < len(self.documents):
                results.append({
                    'content': self.documents[idx],
                    'metadata': self.metadata[idx],
                    'score': 1 - distances[0][i] / 100
                })
        
        return results

class RAGSystem:
    def __init__(self, llm_client, embedding_model="all-MiniLM-L6-v2"):
        self.llm_client = llm_client
        self.vector_store = VectorStore(embedding_model)
    
    def add_knowledge(self, documents: List[Dict]):
        self.vector_store.add_documents(documents)
    
    def retrieve(self, query: str, k: int = 5) -> List[Dict]:
        return self.vector_store.search(query, k)
    
    def build_prompt(self, query: str, retrieved_docs: List[Dict]) -> str:
        context = "\n\n".join([
            f"文档{i+1}：{doc['content']}"
            for i, doc in enumerate(retrieved_docs)
        ])
        
        prompt = f"""基于以下文档回答问题。如果文档中没有相关信息，请说"我不知道"。

文档：
{context}

问题：{query}

回答："""
        
        return prompt
    
    def generate_answer(self, query: str, k: int = 5) -> Dict:
        retrieved_docs = self.retrieve(query, k)
        
        prompt = self.build_prompt(query, retrieved_docs)
        
        response = self.llm_client.chat.completions.create(
            model="gpt-4",
            messages=[{"role": "user", "content": prompt}],
            temperature=0.7
        )
        
        answer = response.choices[0].message.content
        
        return {
            'answer': answer,
            'sources': retrieved_docs,
            'query': query
        }

class AdvancedRAGSystem:
    def __init__(self, llm_client, embedding_model="all-MiniLM-L6-v2"):
        self.llm_client = llm_client
        self.vector_store = VectorStore(embedding_model)
    
    def query_rewriting(self, original_query: str) -> List[str]:
        prompt = f"""请将以下问题改写成更适合检索的形式，生成3个不同的改写版本：

原问题：{original_query}

改写版本："""
        
        response = self.llm_client.chat.completions.create(
            model="gpt-4",
            messages=[{"role": "user", "content": prompt}]
        )
        
        rewritten_queries = response.choices[0].message.content.strip().split('\n')
        
        return [original_query] + rewritten_queries
    
    def hybrid_search(self, query: str, k: int = 5) -> List[Dict]:
        vector_results = self.vector_store.search(query, k * 2)
        
        keyword_results = self.keyword_search(query, k * 2)
        
        combined_results = []
        seen_content = set()
        
        for result in vector_results + keyword_results:
            if result['content'] not in seen_content:
                combined_results.append(result)
                seen_content.add(result['content'])
        
        return combined_results[:k]
    
    def rerank(self, query: str, documents: List[Dict]) -> List[Dict]:
        pairs = [(query, doc['content']) for doc in documents]
        
        scores = []
        for query_text, doc_content in pairs:
            prompt = f"""评估以下文档与问题的相关性，给出0-10的分数：

问题：{query_text}
文档：{doc_content}

相关性分数（0-10）："""
            
            response = self.llm_client.chat.completions.create(
                model="gpt-4",
                messages=[{"role": "user", "content": prompt}]
            )
            
            try:
                score = float(response.choices[0].message.content.strip())
            except:
                score = 0
            
            scores.append(score)
        
        reranked_results = sorted(
            zip(documents, scores),
            key=lambda x: x[1],
            reverse=True
        )
        
        return [doc for doc, score in reranked_results]
    
    def generate_with_citations(self, query: str, k: int = 5) -> Dict:
        retrieved_docs = self.hybrid_search(query, k * 2)
        
        reranked_docs = self.rerank(query, retrieved_docs)[:k]
        
        prompt = self.build_prompt_with_citations(query, reranked_docs)
        
        response = self.llm_client.chat.completions.create(
            model="gpt-4",
            messages=[{"role": "user", "content": prompt}]
        )
        
        answer = response.choices[0].message.content
        
        return {
            'answer': answer,
            'sources': reranked_docs,
            'query': query,
            'citations': self.extract_citations(answer, reranked_docs)
        }
    
    def build_prompt_with_citations(self, query: str, docs: List[Dict]) -> str:
        context = "\n\n".join([
            f"[{i+1}] {doc['content']}"
            for i, doc in enumerate(docs)
        ])
        
        prompt = f"""基于以下文档回答问题，并在回答中引用来源（使用[1]、[2]等标注）。

文档：
{context}

问题：{query}

回答（请标注引用）："""
        
        return prompt
    
    def extract_citations(self, answer: str, docs: List[Dict]) -> List[Dict]:
        citations = []
        for i, doc in enumerate(docs):
            if f"[{i+1}]" in answer:
                citations.append({
                    'id': i+1,
                    'content': doc['content'],
                    'metadata': doc.get('metadata', {})
                })
        
        return citations
    
    def keyword_search(self, query: str, k: int) -> List[Dict]:
        keywords = query.lower().split()
        
        results = []
        for i, doc in enumerate(self.vector_store.documents):
            doc_lower = doc.lower()
            score = sum(1 for keyword in keywords if keyword in doc_lower)
            
            if score > 0:
                results.append({
                    'content': doc,
                    'metadata': self.vector_store.metadata[i],
                    'score': score / len(keywords)
                })
        
        results.sort(key=lambda x: x['score'], reverse=True)
        
        return results[:k]

def test_rag_system():
    from openai import OpenAI
    
    client = OpenAI(api_key="your-api-key")
    
    rag = RAGSystem(client)
    
    documents = [
        {
            'content': 'Python是一种高级编程语言，由Guido van Rossum于1991年创建。',
            'metadata': {'source': 'python_intro.txt'}
        },
        {
            'content': 'Python的设计哲学强调代码的可读性和简洁性。',
            'metadata': {'source': 'python_design.txt'}
        },
        {
            'content': 'Python广泛应用于Web开发、数据科学、人工智能等领域。',
            'metadata': {'source': 'python_usage.txt'}
        }
    ]
    
    rag.add_knowledge(documents)
    
    result = rag.generate_answer("Python是什么时候创建的？")
    
    print(f"问题：{result['query']}")
    print(f"回答：{result['answer']}")
    print(f"来源：{len(result['sources'])}个文档")

if __name__ == "__main__":
    test_rag_system()
```

## 4.3 对话系统开发

### 🎯 对话系统架构

```
┌─────────────────────────────────────────────────────────┐
│         对话系统：多轮交互                               │
└─────────────────────────────────────────────────────────┘

核心组件：
```

```python
from typing import List, Dict
import json
from datetime import datetime

class ConversationManager:
    def __init__(self, max_history=50):
        self.max_history = max_history
        self.conversations = {}
    
    def create_conversation(self, user_id: str) -> str:
        conversation_id = f"{user_id}_{datetime.now().timestamp()}"
        
        self.conversations[conversation_id] = {
            'user_id': user_id,
            'messages': [],
            'context': {},
            'created_at': datetime.now()
        }
        
        return conversation_id
    
    def add_message(self, conversation_id: str, role: str, content: str):
        if conversation_id not in self.conversations:
            raise ValueError("Conversation not found")
        
        message = {
            'role': role,
            'content': content,
            'timestamp': datetime.now()
        }
        
        self.conversations[conversation_id]['messages'].append(message)
        
        if len(self.conversations[conversation_id]['messages']) > self.max_history:
            self.conversations[conversation_id]['messages'] = \
                self.conversations[conversation_id]['messages'][-self.max_history:]
    
    def get_history(self, conversation_id: str) -> List[Dict]:
        if conversation_id not in self.conversations:
            raise ValueError("Conversation not found")
        
        return self.conversations[conversation_id]['messages']
    
    def update_context(self, conversation_id: str, key: str, value):
        if conversation_id not in self.conversations:
            raise ValueError("Conversation not found")
        
        self.conversations[conversation_id]['context'][key] = value
    
    def get_context(self, conversation_id: str) -> Dict:
        if conversation_id not in self.conversations:
            raise ValueError("Conversation not found")
        
        return self.conversations[conversation_id]['context']

class IntentClassifier:
    def __init__(self, llm_client):
        self.llm_client = llm_client
    
    def classify(self, user_input: str) -> Dict:
        prompt = f"""分析用户输入的意图，返回以下信息：

用户输入：{user_input}

请以JSON格式输出：
{
    "intent": "意图类型（如：提问、请求、闲聊、投诉等）",
    "topic": "话题（如：Python学习、产品咨询、技术支持等）",
    "key_info": ["关键信息列表"],
    "emotion": "情感（正面、中性、负面）"
}

输出："""
        
        response = self.llm_client.chat.completions.create(
            model="gpt-4",
            messages=[{"role": "user", "content": prompt}],
            temperature=0.3
        )
        
        try:
            intent_info = json.loads(response.choices[0].message.content)
        except:
            intent_info = {
                'intent': 'unknown',
                'topic': 'unknown',
                'key_info': [],
                'emotion': 'neutral'
            }
        
        return intent_info

class DialogueSystem:
    def __init__(self, llm_client, system_prompt=""):
        self.llm_client = llm_client
        self.conversation_manager = ConversationManager()
        self.intent_classifier = IntentClassifier(llm_client)
        self.system_prompt = system_prompt
    
    def start_conversation(self, user_id: str) -> str:
        conversation_id = self.conversation_manager.create_conversation(user_id)
        
        greeting = self.generate_greeting()
        
        self.conversation_manager.add_message(conversation_id, 'assistant', greeting)
        
        return conversation_id, greeting
    
    def respond(self, conversation_id: str, user_input: str) -> str:
        intent_info = self.intent_classifier.classify(user_input)
        
        self.conversation_manager.update_context(
            conversation_id, 'last_intent', intent_info
        )
        
        self.conversation_manager.add_message(conversation_id, 'user', user_input)
        
        history = self.conversation_manager.get_history(conversation_id)
        context = self.conversation_manager.get_context(conversation_id)
        
        response = self.generate_response(user_input, history, context, intent_info)
        
        self.conversation_manager.add_message(conversation_id, 'assistant', response)
        
        return response
    
    def generate_greeting(self) -> str:
        return "你好！我是AI助手，有什么可以帮助你的？"
    
    def generate_response(self, user_input: str, history: List[Dict], 
                         context: Dict, intent_info: Dict) -> str:
        history_text = "\n".join([
            f"{msg['role']}: {msg['content']}"
            for msg in history[-10:]
        ])
        
        context_text = json.dumps(context, ensure_ascii=False, indent=2)
        
        prompt = f"""系统提示：{self.system_prompt}

对话历史：
{history_text}

上下文信息：
{context_text}

用户意图：{intent_info['intent']}
话题：{intent_info['topic']}
关键信息：{intent_info['key_info']}
情感：{intent_info['emotion']}

用户最新输入：{user_input}

请根据以上信息生成合适的回复。回复应该：
1. 与对话历史连贯
2. 考虑上下文信息
3. 符合用户意图
4. 情感适当

回复："""
        
        response = self.llm_client.chat.completions.create(
            model="gpt-4",
            messages=[{"role": "user", "content": prompt}],
            temperature=0.7
        )
        
        return response.choices[0].message.content
    
    def end_conversation(self, conversation_id: str) -> str:
        history = self.conversation_manager.get_history(conversation_id)
        
        summary = self.summarize_conversation(history)
        
        farewell = "感谢你的交流！再见！"
        
        return farewell, summary
    
    def summarize_conversation(self, history: List[Dict]) -> str:
        history_text = "\n".join([
            f"{msg['role']}: {msg['content']}"
            for msg in history
        ])
        
        prompt = f"""总结以下对话：

对话内容：
{history_text}

请以简洁的方式总结对话的主要内容和结果。

总结："""
        
        response = self.llm_client.chat.completions.create(
            model="gpt-4",
            messages=[{"role": "user", "content": prompt}],
            temperature=0.5
        )
        
        return response.choices[0].message.content
```

```python
from typing import Dict, List, Optional
import json
from datetime import datetime
from dataclasses import dataclass

@dataclass
class CustomerInfo:
    user_id: str
    name: Optional[str] = None
    email: Optional[str] = None
    purchase_history: List[Dict] = None
    preferences: Dict = None

class KnowledgeBase:
    def __init__(self):
        self.products = {}
        self.faqs = {}
        self.policies = {}
    
    def add_product(self, product_id: str, info: Dict):
        self.products[product_id] = info
    
    def add_faq(self, question: str, answer: str, category: str):
        self.faqs[question] = {
            'answer': answer,
            'category': category
        }
    
    def add_policy(self, policy_name: str, content: str):
        self.policies[policy_name] = content
    
    def search_products(self, query: str) -> List[Dict]:
        results = []
        for product_id, info in self.products.items():
            if query.lower() in info['name'].lower() or \
               query.lower() in info['description'].lower():
                results.append({
                    'product_id': product_id,
                    'info': info
                })
        return results
    
    def get_faq_answer(self, question: str) -> Optional[str]:
        for faq_question, faq_info in self.faqs.items():
            if question.lower() in faq_question.lower():
                return faq_info['answer']
        return None
    
    def get_policy(self, policy_name: str) -> Optional[str]:
        return self.policies.get(policy_name)

class CustomerServiceBot:
    def __init__(self, llm_client, knowledge_base: KnowledgeBase):
        self.llm_client = llm_client
        self.knowledge_base = knowledge_base
        self.conversation_manager = ConversationManager()
        
        self.system_prompt = """你是一个专业的客服代表，你的职责是：
        1. 友好、耐心地回答客户问题
        2. 提供准确的产品信息
        3. 解决客户的问题和投诉
        4. 推荐合适的产品
        5. 记录客户反馈
        
        你的特点：
        - 语言礼貌、专业
        - 回答准确、详细
        - 善于倾听和理解
        - 主动提供帮助"""
    
    def handle_message(self, user_id: str, message: str, 
                      customer_info: Optional[CustomerInfo] = None) -> Dict:
        conversation_id = self.get_or_create_conversation(user_id)
        
        intent = self.classify_intent(message)
        
        knowledge = self.retrieve_knowledge(message, intent)
        
        response = self.generate_response(
            message, conversation_id, intent, knowledge, customer_info
        )
        
        self.update_conversation(conversation_id, message, response)
        
        return {
            'response': response,
            'intent': intent,
            'sources': knowledge.get('sources', []),
            'conversation_id': conversation_id
        }
    
    def classify_intent(self, message: str) -> Dict:
        prompt = f"""分析客户消息的意图：

客户消息：{message}

请以JSON格式输出：
{
    "intent": "意图类型（咨询、投诉、购买、退货、技术支持等）",
    "category": "问题类别（产品、价格、服务、技术等）",
    "urgency": "紧急程度（高、中、低）",
    "emotion": "情感（正面、中性、负面）",
    "keywords": ["关键词列表"]
}

输出："""
        
        response = self.llm_client.chat.completions.create(
            model="gpt-4",
            messages=[{"role": "user", "content": prompt}],
            temperature=0.3
        )
        
        try:
            intent = json.loads(response.choices[0].message.content)
        except:
            intent = {
                'intent': 'unknown',
                'category': 'unknown',
                'urgency': 'medium',
                'emotion': 'neutral',
                'keywords': []
            }
        
        return intent
    
    def retrieve_knowledge(self, message: str, intent: Dict) -> Dict:
        knowledge = {'sources': []}
        
        keywords = intent.get('keywords', [])
        
        for keyword in keywords:
            products = self.knowledge_base.search_products(keyword)
            if products:
                knowledge['products'] = products
                knowledge['sources'].extend([p['product_id'] for p in products])
        
        faq_answer = self.knowledge_base.get_faq_answer(message)
        if faq_answer:
            knowledge['faq'] = faq_answer
            knowledge['sources'].append('FAQ')
        
        return knowledge
    
    def generate_response(self, message: str, conversation_id: str,
                         intent: Dict, knowledge: Dict,
                         customer_info: Optional[CustomerInfo]) -> str:
        history = self.conversation_manager.get_history(conversation_id)
        
        history_text = "\n".join([
            f"{msg['role']}: {msg['content']}"
            for msg in history[-5:]
        ])
        
        knowledge_text = ""
        if 'products' in knowledge:
            products_text = "\n".join([
                f"- {p['info']['name']}: {p['info']['description']}"
                for p in knowledge['products']
            ])
            knowledge_text += f"相关产品：\n{products_text}\n"
        
        if 'faq' in knowledge:
            knowledge_text += f"FAQ答案：{knowledge['faq']}\n"
        
        customer_text = ""
        if customer_info:
            customer_text = f"客户信息：\n- 姓名：{customer_info.name}\n- 邮箱：{customer_info.email}\n"
        
        prompt = f"""系统提示：{self.system_prompt}

对话历史：
{history_text}

{customer_text}

客户意图：{intent['intent']}
紧急程度：{intent['urgency']}
情感：{intent['emotion']}

相关知识：
{knowledge_text}

客户最新消息：{message}

请生成合适的回复。回复应该：
1. 友好、专业
2. 针对客户意图
3. 利用相关知识
4. 考虑客户情感

回复："""
        
        response = self.llm_client.chat.completions.create(
            model="gpt-4",
            messages=[{"role": "user", "content": prompt}],
            temperature=0.7
        )
        
        return response.choices[0].message.content
    
    def get_or_create_conversation(self, user_id: str) -> str:
        conversations = [
            cid for cid, conv in self.conversation_manager.conversations.items()
            if conv['user_id'] == user_id
        ]
        
        if conversations:
            return conversations[-1]
        
        return self.conversation_manager.create_conversation(user_id)
    
    def update_conversation(self, conversation_id: str, 
                           user_message: str, assistant_message: str):
        self.conversation_manager.add_message(conversation_id, 'user', user_message)
        self.conversation_manager.add_message(conversation_id, 'assistant', assistant_message)
    
    def handle_complaint(self, message: str, conversation_id: str) -> str:
        prompt = f"""客户投诉处理：

投诉内容：{message}

请按照以下步骤处理：
1. 表示理解和歉意
2. 分析问题原因
3. 提供解决方案
4. 承诺跟进

处理回复："""
        
        response = self.llm_client.chat.completions.create(
            model="gpt-4",
            messages=[{"role": "user", "content": prompt}],
            temperature=0.6
        )
        
        return response.choices[0].message.content
    
    def recommend_product(self, customer_info: CustomerInfo, 
                         query: str) -> List[Dict]:
        prompt = f"""根据客户信息推荐产品：

客户信息：
- 姓名：{customer_info.name}
- 购买历史：{customer_info.purchase_history}
- 喜好：{customer_info.preferences}

客户需求：{query}

请推荐3个合适的产品，以JSON格式输出：
[
    {
        "product_name": "产品名称",
        "reason": "推荐理由",
        "price": "价格",
        "features": ["特点列表"]
    }
]

推荐："""
        
        response = self.llm_client.chat.completions.create(
            model="gpt-4",
            messages=[{"role": "user", "content": prompt}],
            temperature=0.7
        )
        
        try:
            recommendations = json.loads(response.choices[0].message.content)
        except:
            recommendations = []
        
        return recommendations
```
